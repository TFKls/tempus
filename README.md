## Building
```sh
# Generate sources required for development (only required once)
./gradlew genSources 
# Generate required assets via datagen
./gradlew runDatagen
# Build current version to build/libs/tempus-<version>.jar 
./gradlew build
# Build and run current version (for development)
./gradlew runClient
```
